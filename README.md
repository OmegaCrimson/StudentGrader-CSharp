# مشروع تقييم الطلاب — أساسيات C#

This project is a **console-based student grading system** with multi-language support (English/Arabic).  
It allows grade entry for multiple subjects, calculates GPA, provides letter grades, and supports menu-driven navigation.

**هذا المشروع هو نظام تقييم الطلاب على شكل برنامج كونسول، ويدعم اللغتين الإنجليزية والعربية.  
يتيح إدخال الدرجات لعدة مواد، وحساب المعدل التراكمي (GPA)، وعرض التقديرات الحرفية، مع قائمة تنقل بسيطة.**

---

## Features / الميزات

- Multi-subject grade entry / إدخال الدرجات لعدة مواد  
- GPA and average calculation / حساب المعدل التراكمي والمتوسط  
- Accepts English (0–9) and Arabic (٠–٩) digits / يدعم الأرقام الإنجليزية والعربية  
- Export reports / تصدير التقارير  
- Simple menu-driven navigation / قائمة تنقل بسيطة  

---

## How to Use / كيفية الاستخدام

1. Run the app. / شغّل البرنامج.  
2. Select menu items by number. / حدد عناصر القائمة بالأرقام.  
3. Enter student grades when prompted. / أدخل درجات الطلاب عند الطلب.  
4. Use the **Settings** menu to change language. / استخدم قائمة **الإعدادات** لتغيير اللغة.

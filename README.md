# C# Foundational — Student Grading App
# مشروع تقييم الطلاب — أساسيات C#

Hi! I'm **Mohamed Gonem** / مرحباً! أنا **محمد غنيم**.  
This is a simple console-based student grading app I developed to practice C# fundamentals.  
هذا برنامج كونسول لتقييم درجات الطلاب طورته لأتمرن على أساسيات لغة #C.

---

## Features / الميزات
- Enter grades for multiple subjects easily  
- إدخال درجات للعديد من المواد بسهولة
- Automatic calculation of Average and GPA  
- حساب المتوسط وGPA تلقائياً
- Supports both English (0–9) and Arabic (٠–٩) digits  
- يدعم الأرقام الإنجليزية (0–9) والعربية (٠–٩)
- Simple and clean menu-driven navigation  
- قائمة تنقل بسيطة وواضحة بالأرقام
- Settings: change language and back option  
- الإعدادات: تغيير اللغة والعودة للقائمة

---

## How to Use / كيفية الاستخدام
### Windows:
1. Run `StudentGradingApp.exe`.  
   شغّل الملف `StudentGradingApp.exe`.
2. Navigate menus with numbers or Arabic digits.  
   استخدم الأرقام أو الأرقام العربية للتنقل في القوائم.

### Android:
1. Install `StudentGradingApp.apk` on your device.  
   قم بتثبيت الملف `StudentGradingApp.apk` على جهازك.
2. Open the app and follow the prompts.  
   افتح التطبيق واتبع التعليمات على الشاشة.
